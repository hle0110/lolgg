A running log of changes to the site.

July 21, 2026

Added a friendly error banner instead of a blank page if something breaks, an offline notice, and accessibility improvements across search, navigation, and filters. Added optional browser notifications for when your favorited teams go live, an Add to Calendar button on upcoming matches, Copy Link buttons on match and tournament pages, a compact view toggle for the schedule list, and this changelog page.

July 19, 2026

Removed the live tournament banner from the homepage. Trimmed the timezone picker down to the most-used timezones and moved it next to the clock. Replaced the email contact link with a proper contact form so the site's inbox address isn't exposed in the page source. Cleaned up wording across the terms, privacy, and credits pages.

July 18, 2026

Fixed the Ongoing tab so it only shows matches from the site's supported leagues instead of every league Riot tracks. Fixed a bug where a match that started later than scheduled wasn't showing up as live.

July 17, 2026

Added a live countdown timer for upcoming matches, sped up schedule loading, and added an official Huya stream link for LPL matches. Removed all code comments ahead of moving to a fresh repository.

Earlier

Initial launch: live scores, full match schedule, tournament brackets and standings, VODs, official stream links, team and league search, and favoriting.
